**Cannot use this profile:** no API Key is set for the OpenAI-compatible endpoint, and `OPENAI_API_KEY` is not in the environment.

Add a key in **Settings** and save, or switch to the built-in **Local Ollama (default)** profile.

_No request was sent to the model service._

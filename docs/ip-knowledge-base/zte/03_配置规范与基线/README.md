# 03_配置规范与基线

## 定位
记录公司内部标准配置模板和最佳实践，解决"应该怎么配"的问题。这是连接理论与实战的桥梁。

## 内容要求
- **配置基线**：各协议的标准配置模板（含推荐参数和避坑指南）
- **配置规范**：跨域VPN、MPLS等复杂场景的标准化部署方案
- **面积规划模板**：OSPF区域划分、IS-IS层级设计等网络规划参考

## 与01_协议排障逻辑树的区别
- 01文件夹讲"排查思路"（If...Then...决策树）

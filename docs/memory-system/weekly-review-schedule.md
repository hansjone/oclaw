# Weekly Review Schedule

Run once per week, fixed to the same weekday and time block.

- Recommended slot: Friday 17:30-18:00 (or any stable 30-minute slot)
- Hard rule: do not skip two weeks in a row

## Inputs

- `Inbox/` captured items
- New or edited notes in `Resources/`
- Active materials in `Projects/` and `Areas/`
- SRS backlog and daily new-card load

## Execution

Follow `templates/weekly-review.md` in order:

1. Inbox Zero
2. Linking and Structure
3. Archive and Cleanup
4. SRS Adjustment

## Outputs

- Updated links/backlinks in new atomic notes
- Archived inactive items moved to `Archives/`
- Next week focus topics (top 3)
- Adjusted daily new-card limit (<= 10/day)

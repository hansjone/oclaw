# Daily Memory Routine (10-20 minutes)

- Date: `<YYYY-MM-DD>`
- Focus topic: `<project/area/resource>`

## 1) Capture (3-5 items)

- [ ] Capture item 1
- [ ] Capture item 2
- [ ] Capture item 3
- [ ] Capture item 4 (optional)
- [ ] Capture item 5 (optional)

## 2) Distill (1-3 atomic notes)

- [ ] Rewrite in own words
- [ ] Keep one note for one claim
- [ ] Add at least one link to an existing note

## 3) Recall (3-10 cards)

- [ ] Convert high-value notes to Q/A or cloze
- [ ] Keep each answer short and testable
- [ ] Tag cards by PARA context

## 4) Express (1 output)

- [ ] Produce one concrete output:
  - summary paragraph / answer draft / code note

## End-of-Day Check

- [ ] No unprocessed high-value item left in `Inbox`
- [ ] New cards count is sustainable for tomorrow

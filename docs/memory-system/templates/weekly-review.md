# Weekly Review Template (30 minutes)

- Review date: `<YYYY-MM-DD>`
- Owner: `<name>`

## 1) Inbox Zero (10 minutes)

- [ ] Process all notes from `Inbox`
- [ ] Convert valuable items into atomic notes
- [ ] Delete low-value noise

## 2) Linking and Structure (8 minutes)

- [ ] Ensure each new atomic note links to at least one existing note
- [ ] Add missing backlinks for key notes
- [ ] Move notes to correct PARA container

## 3) Archive and Cleanup (6 minutes)

- [ ] Archive inactive project notes to `Archives`
- [ ] Remove duplicate notes or cards
- [ ] Mark stale resources for later refresh

## 4) SRS Adjustment (6 minutes)

- [ ] Check review backlog and adjust new card limit if needed
- [ ] Keep next week new cards <= 10/day
- [ ] Pick top 3 focus topics for next week

## Weekly Output

- One paragraph insight:
  - `<what changed in understanding this week>`
- One practical application next week:
  - `<where to apply in writing/decision/coding>`

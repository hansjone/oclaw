本目录用于 **公共 skills**。

约定：
- 路径：`oclaw/runtime/skills/_workspace/public/<skill_name>/SKILL.md`
- 放在这里的 skill **默认对所有人可用**，不需要在 Admin 里做角色绑定。

注意：
- 仍会受 skill 本身的 `disable_model_invocation`、租户禁用列表等策略影响。

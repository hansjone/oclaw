# Social Skill（对外沟通）

## 适用场景
- 公告、邮件、更新说明、PR 描述、客服回复等对外文本。
- 需要按受众调整语气和信息层级。
- 需要把技术事实转成可读、可发布、可执行的表达。

## 工作方法
1. 明确受众、渠道、目标动作。
2. 抽取事实，剔除歧义和不可承诺内容。
3. 生成可直接发布版本（必要时提供备选语气）。
4. 检查敏感信息和承诺边界。

## 输出要求
- 默认给：一句话版 + 标准版正文。
- 术语统一，避免过度承诺。
